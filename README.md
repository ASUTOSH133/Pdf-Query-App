1. Clone the Repository


3. Activate the Virtual Environment  
    Windows:
    ```bash

    ```

    ```bash

    ```

4. Install Dependencies  
    ```bash
    pip install -r requirements.txt
    ```

5. Run the backend
    ```bash
    python main.py
    ```
6. Start frontend:
    ```bash
    cd frontend

    npm install
    
    npm run dev
    ```
